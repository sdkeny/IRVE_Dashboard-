À propos — Sources de données

Bornes de recharge

IRVE consolidé — data.gouv.fr : base nationale géolocalisée avec puissance, type de prise, opérateur et statut. Point de départ obligatoire.

Demande potentielle

IRIS — INSEE : population, revenus médians, part de résidences avec voiture, densité de ménages. Proxy le plus fin pour estimer la demande.

Immatriculations VE — SDES : données par département, mise à jour mensuelle.

Usage du sol

BD TOPO — IGN : bâtiments, routes, réseau viaire. Indispensable pour l'accessibilité physique.

OCS GE — Île-de-France : occupation du sol à grande échelle pour le zonage fonctionnel.

Attracteurs de mobilité

OpenStreetMap (Overpass Turbo) : centres commerciaux, parkings, zones d'activité, gares.

GTFS — Île-de-France Mobilités : réseau TC, nœuds multimodaux.

Accessibilité routière

BD CARTO / OSM : réseau viaire. API OpenRouteService : calcul d'isochrones.

Références territoriales

IRIS, communes, EPT Grand Paris — data.gouv.fr et Géoportail de l'Urbanisme.
